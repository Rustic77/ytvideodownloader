API ENDPOINTS
================================================================================

POST /api/info
--------------
Request:  { "url": "https://youtube.com/watch?v=..." }
Response: { "success": true, "title": "...", "duration": 180, ... }

POST /api/download
------------------
Request:  { "url": "https://youtube.com/...", "quality": "1080p" }
Response: { "success": true, "job_id": "uuid-here" }

GET /api/status/{job_id}
------------------------
Response: { "status": "completed", "progress": 100, "token": "uuid" }

GET /api/file/{token}
---------------------
Response: File download (video/mp4)

GET /health
-----------
Response: { "status": "healthy" }
